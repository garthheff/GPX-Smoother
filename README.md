# GPS Smoother
A simple browser-based tool to clean noisy GPX distance and elevation data. 

# Link
https://garthheff.github.io/GPS-Smoother/

# Why
My raw GPX files showed inflated distance and elevation, while Garmin and Strava displayed corrected values.
Since their cleanup isn’t accessible or exportable, I built this to fix GPX data outside those platforms.

# Features
* Compare original vs fixed tracks on a map
* Combined elevation graph
* Adjustable smoothing (distance and elevation)
* Fix log showing what changed
* Export cleaned _fixed.gpx

# Usage
* Open the page
* Upload a GPX file
* Adjust smoothing
* Download the cleaned GPX
